What is this project about?
This project is a collection of games inspired by classic Operations Research problem. For each game the user will need to find the optimal solution for a collection of increasingly difficult instances. For each game there will be an option to automatically compute the optimal solution, while getting an explanation of the underlying algorithm.

Technological stack:
The whole project has to be able to run in the browser to faciliate the deployment of it. It should be playable on computers, tablets and ideally smartphones has well. It should offer a dark mode and a light mode.

## How to play
Simply open `index.html` in any modern web browser.

## Games
- **Shortest Path**: Find the optimal path from source to target in a weighted graph.
